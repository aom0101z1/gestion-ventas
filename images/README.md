# Carpeta de Imágenes - Ciudad Bilingüe

Esta carpeta contiene las imágenes utilizadas en el sistema.

## Archivos Requeridos para Paz y Salvo:

1. **logo.png** - Logo de Ciudad Bilingüe
   - Tamaño recomendado: 150x150 px
   - Formato: PNG con fondo transparente

2. **firma.png** - Firma del director/administrador
   - Tamaño recomendado: 200x80 px
   - Formato: PNG con fondo transparente

## Instrucciones:

1. Sube tus archivos `logo.png` y `firma.png` a esta carpeta
2. Asegúrate de que los nombres sean exactamente como se indica
3. Usa formato PNG para mejor calidad
4. Los archivos se cargarán automáticamente en el certificado Paz y Salvo
